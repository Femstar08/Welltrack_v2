## Codebase Patterns
- Use cmd.exe /c to run Flutter commands on WSL2
- Must pass --dart-define-from-file=.env for Supabase credentials
- All navigation uses GoRouter — NEVER use Navigator.pushNamed()
- All Supabase calls through repository classes
- AI calls through ai_orchestrator_service.dart only
- Freemium gates use currentPlanTierProvider (FutureProvider)
- PrescriptionEngine is static class — day type mapping in today_nutrition_provider.dart
- RecipeGenerationNotifier is in recipe_generation_provider.dart
- PantryItemEntity has: name, category, quantity, unit, isAvailable, expiryDate
- pantryRepositoryProvider for pantry data access
- Hive for local settings (use Hive.box('settings'))
